# GateLink v0.2 · 买家任务拆分

## Phase 1 · 认证体验优化

- 注册成功后不再展示完整 API Key
- 注册成功后主路径改为引导去登录
- 登录成功后自动跳转到买家后台默认首页

## Phase 2 · 买家后台壳子重构

- 顶部横向导航改为左侧导航
- 增加顶部栏与右上角个人中心
- 明确默认首页信息架构

## Phase 3 · 广场

- 广场首页
- 搜索与筛选
- 资源列表与详情
- 购买入口

## Phase 4 · 我的购买

- 已购资源列表
- 状态、时间、唯一 ID 管理
- 到期信息与详情页

## Phase 5 · 账户与经营页完善

- API Key 管理增强
- 充值与账单整合
- 用量分析增强
- 个人中心
