# Seller Spec v0.3

## 1. 卖家端目标

让卖家通过钱包登录平台，绑定 HL 收款地址，配置可售资源，并通过平台 usage 结算获得收益。

## 2. 主要页面

- `Wallet Sign-In`
- `Seller Dashboard`
- `Resources`
- `Pricing`
- `Payout Wallet`
- `Earnings`
- `Settlements`
- `Settings`

## 3. 核心功能

### 3.1 钱包注册/登录

- 卖家使用钱包签名创建/登录账户
- 默认登录钱包可作为初始身份钱包

### 3.2 收款地址绑定

- 默认支持使用登录钱包作为收款地址
- 支持改绑地址，但必须重新签名确认
- 改绑后可设置冷却期和风控提醒

### 3.3 资源管理

- 上架资源
- 配置支持模型
- 配置价格与分成
- 设置启用/停用状态

### 3.4 收益中心

- 待结算收益
- 累计已结算
- 平台佣金拆账说明
- 收益趋势

### 3.5 结算

- 卖家达到门槛后可申请结算
- 平台可按周期自动结算
- 结算完成后展示链上哈希与状态

## 4. 页面级信息规范

### Dashboard

- 待结算收益
- 今日调用量
- 当前在线资源数
- 近期结算状态

### Resources

- 资源名称
- 支持模型
- 单价/倍率
- 状态
- 最近错误率

### Settlements

- 申请时间
- 结算金额
- 打款状态
- 链上哈希

## 5. 关键规则

- 卖家原始资源 Key 不向买家暴露
- 卖家收益来自 usage 完成后的平台拆账
- 链上打款只发生在结算时，不随每次请求实时打款
