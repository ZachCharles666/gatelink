# Admin Spec v0.3

## 1. 管理端目标

让平台管理员能够监控充值、退款、卖家结算、风险事件和账本对账，确保 Hyperliquid 资金流与平台内账一致。

## 2. 主要页面

- `Overview`
- `Deposits`
- `Refunds`
- `Settlements`
- `Risk Control`
- `Reconciliation`
- `Audit Logs`

## 3. 核心功能

### 3.1 总览

- 平台链上余额
- 买家 Credits 总额
- 卖家待结算总额
- 当日充值、退款、结算数量

### 3.2 充值监控

- 充值意图状态
- 链上到账状态
- 异常充值
- 重复充值识别

### 3.3 退款处理

- 退款申请列表
- 大额人工审核
- 失败退款重试

### 3.4 卖家结算台

- 待结算卖家列表
- 自动结算结果
- 失败打款与重试

### 3.5 风控

- 高频退款
- 大额异常充值
- 收款地址改绑
- 疑似盗号或异常消耗

### 3.6 对账

- 链上平台余额
- 平台 Credits 负债
- 卖家待结算负债
- 日对账结果与差异告警

## 4. 关键规则

- 管理端是异常处理与风控入口，不是买家主流程依赖
- 对账必须每日执行
- 所有关键动作需要审计日志
