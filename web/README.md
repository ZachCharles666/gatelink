# GateLink Web

Seller + buyer console built with Next.js 14 + TypeScript.

Current docs:
- `docs/README.md`
- `docs/product/CURRENT.md`
- `docs/product/v0.3/README.md`

## Structure

```
web/
├── app/
│   ├── seller/   # Seller console pages
│   ├── buyer/    # Buyer console pages
│   └── admin/    # Admin panel
├── components/   # Shared UI components
└── lib/api/      # API client wrappers
```

## Run

```bash
npm install
npm run dev
```
